Ouvrir ce fichier dans un navigateur. Pour obtenir un PDF: Ctrl+P → Enregistrer en PDF. Pour récupérer les diagrammes en image: clic droit sur le diagramme (SVG) → enregistrer.

Architecture (résumé)

  • UI: JavaFX + FXML + CSS
  • DB: MySQL via JDBC + pool HikariCP
  • Rôles: ADMIN / SECRETAIRE / MEDECIN / PATIENT
  • Fonctions: Patients, médecins, spécialités, disponibilités, rendez-vous, dossiers médicaux, ordonnances

Où le cours “Threads & Collections” est utilisé ?

  • Threads: AppExecutors (thread pool) + Task JavaFX dans RendezVousController pour chargements DB asynchrones.
  • Collections: ObservableList (List UI), HashMap (Map index par id) dans RendezVousController.
  • Streams: DatabaseInitializer utilise Collectors.joining et Arrays.stream.
